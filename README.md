# 🧠 LLM UI Interface (NotebookLLM-inspired)

A modern and responsive UI for an LLM-powered document chat interface inspired by NotebookLLM. This project focuses on delivering a seamless experience for uploading documents and interacting with an AI chat system.

---

## 🚀 Features

### 📁 File Upload System
- Upload **single or multiple files**
- Real-time **upload progress tracking**
- Ability to **cancel uploads**
- Clean and intuitive UI

### 💬 Chat Interface
- Chat UI inspired by ChatGPT
- Smooth and responsive conversation layout
- User & assistant message bubbles
- Scrollable chat view

### 🎨 UI/UX
- Built with **React.js**
- Styled using **Tailwind CSS**
- Modern, minimal, and reusable component design

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **HTTP Client:** Axios  

---

## 📦 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/AnudeepthiKolagani/ai-doc-assistant.git
cd ai-doc-assistant
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the Development server
```bash
npm start
```

### 4. Open in browser
```bash
http://localhost:3000
```
